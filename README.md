# TECCI: Tricky Edits of Collected and Curated Images

This repository contains the data for the paper:

**TECCI: A Benchmark for Evaluating Image Editing with Tricky Instructions**

## Overview

TECCI is a new image editing benchmark designed to systematically evaluate the capabilities of state-of-the-art text-guided image editing methods.

## Dataset

TECCI consists of **7,550 pairs** of images and edit instructions, spanning **7 image categories**:

| Category | Description |
|---|---|
| Text | Images containing text elements |
| Art | Artistic images and artworks |
| Animal | Images featuring animals |
| Clock | Clock and time-related images |
| Vehicle | Images of vehicles |
| Architecture | Architectural images |
| Nature | Natural scenes and landscapes |

These categories are specifically chosen to target the weaknesses of existing editing methods.

### Edit Instructions

- **Auto-generated instructions**: Generated by Gemini, covering **5 edit types** per source image
- **Manually curated instructions**: A smaller set of **530** challenging, hand-written edit instructions
